🎯 Focus 효과 테스트

✨ 새로운 Focus 효과 기능

  • Focus 시 부드러운 글로우 효과와 상승 애니메이션
  • Hover 시 미묘한 프리뷰 효과
  • 타이핑 중 배경색 그라데이션과 향상된 그림자
  • 전송 시 성공 색상으로 펄스 애니메이션
  • 실시간 문자 수 피드백 (경고 색상 변화)
  • 부드러운 플레이스홀더 이동 효과
  • 커서 색상 변경 (파란색)
Focus 효과 테스트입니다! 아래 입력창을 클릭하고 다양한 효과를 체험해보세요.

테스트 방법:
• 입력창을 클릭해서 Focus 효과를 확인하세요
• 마우스를 올려서 Hover 효과를 확인하세요
• 텍스트를 입력하면서 타이핑 효과를 확인하세요
• Enter 키를 눌러서 전송 펄스 효과를 확인하세요
• 버튼으로 긴 텍스트를 추가해서 문자 수 경고를 확인하세요
